1. Производится вызов инициализационного сервлета
	При загрузке сервлета
  1.1 Производится вызов абстрактного метода в котором производится 
регистрация сервиса обслуживания канала
  1.2 Производится инициализация сервиса
        Функция сервиса
  1.3 Создается новый канал и возвращается его идентификатор 
      вызывающей стороне

2. Клиент обращается за данными по HTTP к специализированному сервлету или JSP
    2.1 Передается идентификатор канала
    по этому идентификатору в сервлете или JSP возвращается канал
    2.2. Из канала берутся данные и отправляются клиенту до тех пор пока
либо истечет таймаут, либо из канала получено конечное сообщение.
При истечении таймаута клиент снова пытается соединится к сервлету, если 
нет тогда канал клиента сообщает об ошибке выше стоящему уровню.

3. Клиент обращается для передачи данных к специлизированному сервлету или JSP
	Данные клиентом передаются пакетом за одно обращение, после 
   этого клиент закрывает соединение, для нового обращения производится 
   повторное соединение (сейчас сделано так из-за возможностей хостинга).

       Как в пункте 2, в сервлете или JSP производится прием идентификатора 
       канала по которому получается канала и в этот канала записывается пакет
       данных. 


			Каналы

Данные передаваемые сервлетам и получаемые от сервлета берутся из канала
сервлета. Серверная программа выполняющая бизнес логику так же получает данные 
из канала. Соедиение этих каналов осуществляет ServletChannel, который 
представляет из себя своеобразный четырех полюсник, одна сторона которого 
принадлежит сервлетам, другая программе бизнес логики.

Размещение этого четырех полюсника производится в инициализационном сервлете
там же производится и запуск потока обслуживающего бизнес логику канала (т.е. 
запуск программы собственно которой предназначены данные от клиента)



			Манагеры

ServletChannelManagerServletChannelManager - снглентон который является хранилищем размещенных 
каналов содержит хеш таблицу каналов. так же содержит поток монитора каналов
который осуществляет очистку неиспользуемых или замерших каналов.
